# SowTempo Documentation

Official user documentation for [SowTempo](https://sowtempo.com), a privacy-focused vegetable garden planning and management application.

## View the Documentation

**Live Documentation:** [https://mattnudi.github.io/SowTempoDocs/](https://mattnudi.github.io/SowTempoDocs/)

## About SowTempo

SowTempo helps thoughtful gardeners plan and manage their vegetable gardens with:

- **Visual Garden Designer** — Design beds with intuitive canvas and proper plant spacing
- **Smart Planting Schedule** — Personalized planting dates based on location and frost dates
- **Plant Database** — Nearly 4,000 vegetable varieties with detailed growing information
- **GDD Tracking** — Predict plant maturity using Growing Degree Days
- **Harvest Tracking** — Record harvests and analyze garden productivity
- **Privacy First** — All data stays on your device, no accounts required

## Local Development

This documentation is built with [Docusaurus](https://docusaurus.io/).

```bash
# Install dependencies
npm install

# Start local development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
