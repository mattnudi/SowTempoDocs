# Garden Journal

The Garden Journal is your garden's diary — a place to record observations, document progress with photos, and keep notes for future reference.

![Garden Journal](/img/screenshots/journal-overview.png)

## Creating Journal Entries

### New Entry

1. Click **+ New Entry**
2. Fill in entry details
3. Add photos if desired
4. Link to plantings or beds
5. Click **Save**

### Entry Fields

| Field                | Description                     |
| -------------------- | ------------------------------- |
| **Title**            | Brief headline for the entry    |
| **Date**             | When this observation was made  |
| **Content**          | Full text of your notes         |
| **Tags**             | Categories for organization     |
| **Photos**           | Images documenting the entry    |
| **Linked Plantings** | Connect to specific plants      |
| **Linked Beds**      | Connect to specific garden beds |

![New Journal Entry](/img/screenshots/journal-new-entry.png)

### Writing Content

The content field supports basic formatting:

- Paragraphs
- Bullet lists
- Numbered lists

Write as little or as much as you want. Quick notes and detailed observations are both valuable.

## Adding Photos

### Uploading Images

1. Click **Add Photo** in the entry editor
2. Select images from your computer
3. Photos appear in the gallery

### Photo Features

- **Multiple photos** — Add several images per entry
- **Full-size view** — Click to enlarge
- **Remove photos** — Click the remove button on any photo

Photos are stored locally with your data.

![Journal Photos](/img/screenshots/journal-photos.png)

:::tip
Take photos consistently — weekly "garden walk" photos create a valuable visual timeline.
:::

## Using Tags

Tags help organize and find entries later.

### Adding Tags

1. Click in the **Tags** field
2. Type a tag name
3. Press Enter to add
4. Repeat for multiple tags

### Suggested Tags

- **Weather** — Frost, heat wave, rain
- **Activities** — Planting, harvesting, weeding
- **Issues** — Pests, disease, damage
- **Milestones** — First harvest, last planting
- **Observations** — Growth, flowering, fruiting

### Managing Tags

- Click the **x** on a tag to remove it
- Tags are automatically saved
- Reuse existing tags when possible for consistency

## Linking to Plantings and Beds

Connect entries to specific items in your garden:

### Linking a Planting

1. Click **Link Planting**
2. Select from your current season's plantings
3. The planting is associated with this entry

Linked plantings show a reference to the entry in their detail view.

### Linking a Bed

1. Click **Link Bed**
2. Select a bed from Garden Layout
3. The bed is associated with this entry

This is useful for bed-specific notes like soil amendments or layout changes.

## Browsing Entries

### Entry List

The journal displays entries chronologically, newest first:

![Journal List](/img/screenshots/journal-list.png)

Each entry shows:

- Title
- Date
- Preview of content
- Tags
- Photo thumbnails

### Searching

Use the search box to find entries by:

- Title text
- Content text
- Tag names

### Filtering

Click tags in the sidebar to filter entries:

- Click a tag to show only entries with that tag
- Click again to remove the filter
- Multiple tags show entries matching any selected tag

### Filter by Planting

1. Click **Filter by Planting**
2. Select a planting
3. View all entries linked to that planting

### Filter by Bed

1. Click **Filter by Bed**
2. Select a garden bed
3. View all entries linked to that bed

## Viewing Entry Details

Click any entry to see the full content:

- Complete text
- Full-size photos
- All linked items
- Weather snapshot (if captured)

## Editing Entries

1. Open the entry
2. Click **Edit**
3. Make changes
4. Click **Save**

You can edit any field, add/remove photos, and update links.

## Deleting Entries

1. Open the entry
2. Click **Delete**
3. Confirm deletion

Deleted entries are permanently removed.

## Weather Snapshots

When you create an entry, SowTempo captures current weather:

- Temperature
- Conditions
- Precipitation

This context helps you correlate observations with weather conditions.

## Journal Ideas

### Regular Entries

- **Weekly garden walk** — Overview of garden status
- **Planting records** — What you planted today
- **Harvest notes** — Quality and quantity observations
- **Weather events** — Significant weather and its effects

### Issue Documentation

- **Pest sightings** — Photos and identification
- **Disease symptoms** — What you observed
- **Damage assessment** — After storms or extreme weather
- **Treatment applied** — What you did about problems

### Planning Notes

- **Variety impressions** — How new varieties performed
- **Layout ideas** — Changes to consider
- **Lessons learned** — What to do differently
- **Season review** — End-of-season summary

## Tips for Journaling

1. **Be consistent** — Regular entries beat sporadic detailed ones
2. **Take photos** — Visual records are invaluable
3. **Use tags** — Makes finding entries easier
4. **Link items** — Connect to plantings for context
5. **Review annually** — Read last year's journal before planning
