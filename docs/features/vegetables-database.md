# Vegetables Database

SowTempo includes a comprehensive database of nearly 4,000 vegetable varieties, complete with growing information, spacing requirements, and frost tolerance data.

![Vegetables Database](/img/screenshots/vegetables-overview.png)

## Searching

Use the search box to find specific vegetables or varieties:

- Search by common name: "tomato"
- Search by variety: "Cherokee Purple"
- Search by characteristic: "heirloom"

Results show matches across all categories.

## Filtering

Narrow down results with filters:

### Frost Tolerance

Filter by cold hardiness:

| Level           | Description                                |
| --------------- | ------------------------------------------ |
| **Very Tender** | No frost tolerance (tomatoes, peppers)     |
| **Tender**      | Damaged by light frost (beans, squash)     |
| **Half Hardy**  | Tolerates light frost (lettuce, brassicas) |
| **Hardy**       | Survives hard frost (kale, spinach)        |
| **Very Hardy**  | Overwinters in cold climates (garlic)      |

### Days to Maturity

Use the slider to filter by harvest timeline:

- Quick crops (30-45 days)
- Medium season (45-75 days)
- Long season (75+ days)

### Zone Compatibility

Filter for plants suited to your USDA Hardiness Zone. This shows varieties that can be grown successfully in your climate.

### Custom Plants Only

Toggle to show only your custom-added varieties.

![Filters](/img/screenshots/vegetables-filters.png)

## Vegetable Details

Click any vegetable to see detailed information:

![Vegetable Detail](/img/screenshots/vegetable-detail.png)

### Basic Information

| Field               | Description                               |
| ------------------- | ----------------------------------------- |
| **Common Name**     | Standard name (e.g., "Tomato")            |
| **Scientific Name** | Latin name (e.g., _Solanum lycopersicum_) |
| **Plant Family**    | Botanical family (e.g., Solanaceae)       |
| **Category**        | SowTempo category                         |

### Growing Information

| Field                         | Description                             |
| ----------------------------- | --------------------------------------- |
| **Days to Maturity**          | Time from planting to harvest           |
| **Frost Tolerance**           | Cold hardiness level                    |
| **Preferred Planting Method** | Seed indoors, direct sow, or transplant |
| **Growing Season**            | Spring, summer, fall, or year-round     |

### Spacing Requirements

| Field             | Description             |
| ----------------- | ----------------------- |
| **Plant Spacing** | Distance between plants |
| **Row Spacing**   | Distance between rows   |
| **Depth**         | How deep to plant seeds |

### Growing Notes

Additional tips and information about growing this vegetable, including:

- Soil preferences
- Sunlight requirements
- Watering needs
- Common issues

## Variety Information

Click into a specific variety for cultivar-specific data:

![Variety Detail](/img/screenshots/variety-detail.png)

### Variety Properties

- **Variety Name** — Cultivar name (e.g., "Cherokee Purple")
- **Days to Maturity Override** — If different from base vegetable
- **Heirloom Status** — Whether it's an heirloom variety
- **Growth Habit** — Determinate, indeterminate, bush, etc.
- **Color/Size** — Physical characteristics
- **Flavor Notes** — Taste description

### Variety Actions

- **Add to Schedule** — Create a planting with this variety
- **Add to Favorites** — Save for quick access

## Custom Plants

Add your own vegetables and varieties that aren't in the database:

### Adding a Custom Plant

1. Click **+ Add Custom Plant**
2. Fill in the required fields:
   - Plant name
   - Category
   - Days to maturity
   - Frost tolerance
   - Plant spacing
3. Add optional information:
   - Growing notes
   - Preferred planting method
   - Row spacing
4. Click **Save**

![Add Custom Plant](/img/screenshots/vegetables-add-custom.png)

### Managing Custom Plants

- **Edit** — Modify any custom plant you've created
- **Delete** — Remove a custom plant
- **Filter** — Use "Custom Plants Only" to see just your additions

:::tip
Custom plants work exactly like built-in varieties. You can add them to plantings, track harvests, and include them in your garden layout.
:::

## Using Vegetables in Your Garden

### Adding to Planting Schedule

From any vegetable detail page:

1. Click **Add to Schedule**
2. Select variety (if applicable)
3. Choose planting method
4. Review calculated dates
5. Save the planting

### Adding to Garden Layout

When adding a plant placement in Garden Layout:

1. Select the **Plant** tool
2. Click in a bed
3. Search for the vegetable
4. Select variety
5. Configure spacing and count

## Tips for Using the Database

1. **Start with filters** — Narrow down by frost tolerance for your zone
2. **Compare varieties** — Look at days to maturity when choosing
3. **Check spacing** — Use database spacing for accurate layout planning
4. **Read the notes** — Growing notes often contain valuable tips
5. **Add custom plants** — Track any variety not in the database
