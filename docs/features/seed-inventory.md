# Seed Inventory

Track your seed collection with the Seed Inventory. Know what you have, when it expires, and where you got it.

![Seed Inventory](/img/screenshots/inventory-overview.png)

## Adding Seeds

### New Seed Entry

1. Click **+ Add Seeds**
2. Fill in seed details
3. Click **Save**

### Seed Fields

| Field               | Required | Description                      |
| ------------------- | -------- | -------------------------------- |
| **Vegetable**       | Yes      | Type of seed                     |
| **Variety**         | No       | Specific cultivar                |
| **Quantity**        | Yes      | Number of seeds or packets       |
| **Unit**            | Yes      | Seeds, Packets, Grams, or Ounces |
| **Supplier**        | No       | Where purchased                  |
| **Purchase Date**   | No       | When seeds were bought           |
| **Expiration Date** | No       | When seeds expire                |
| **Lot Number**      | No       | Supplier's batch/lot code        |
| **Notes**           | No       | Growing tips, germination notes  |

![Add Seed](/img/screenshots/inventory-add.png)

### Linking to Database

When you select a vegetable, SowTempo can link to the plant database:

- Shows growing information
- Suggests spacing and timing
- Connects to planting schedules

## Viewing Inventory

### Inventory List

The main view shows all seeds with:

- Vegetable and variety
- Quantity remaining
- Expiration status
- Supplier name

![Inventory List](/img/screenshots/inventory-list.png)

### Quick Filters

Filter buttons show:

| Filter            | Description                         |
| ----------------- | ----------------------------------- |
| **All Seeds**     | Everything in inventory             |
| **Low Stock**     | Items with ≤10 units (with count)   |
| **Expiring Soon** | Expires within 30 days (with count) |

### Search

Search by:

- Vegetable name
- Variety name
- Supplier name

## Stock Alerts

### Low Stock

Seeds with quantity ≤10 show a **Low Stock** warning. This reminds you to reorder before you run out.

### Expiring Soon

Seeds expiring within 30 days show an **Expiring Soon** warning. Plan to use these seeds first.

![Stock Alerts](/img/screenshots/inventory-alerts.png)

## Adjusting Quantity

### Quick Adjust

Use the +/- buttons to quickly update quantity:

- Click **+** to increase
- Click **-** to decrease

### Full Edit

1. Click **Edit** on the seed entry
2. Change the quantity field
3. Save

### When to Adjust

Update quantity when you:

- Plant seeds from this stock
- Receive new seeds
- Discard old or damaged seeds

## Tracking Seed Usage

### Usage History

View how seeds have been used:

1. Click **View Usage History** on a seed entry
2. See linked plantings
3. View dates and quantities used

This shows:

- Which plantings used this seed
- How many were planted
- Germination success (if tracked)

### Linking to Plantings

When creating a planting, you can link to inventory:

1. In the planting editor, select **Seed Source**
2. Choose from your inventory
3. Quantity is automatically adjusted

## Seed Storage Tips

### Optimal Storage

- **Cool** — 40-50°F (4-10°C) is ideal
- **Dry** — Low humidity prevents mold
- **Dark** — Light degrades seeds
- **Airtight** — Prevents moisture absorption

### Storage Locations

Common storage locations to track:

- Refrigerator (vegetable drawer)
- Freezer (for long-term)
- Cool basement or garage
- Climate-controlled room

### Seed Viability

Typical seed life with proper storage:

| Seeds                     | Viability |
| ------------------------- | --------- |
| Onion, Parsnip, Parsley   | 1-2 years |
| Corn, Pepper, Lettuce     | 2-3 years |
| Beans, Carrot, Spinach    | 3-4 years |
| Tomato, Brassicas, Squash | 4-5 years |
| Cucumber, Melon           | 5+ years  |

## Managing Suppliers

Track where you buy seeds:

### Recording Suppliers

Include in each seed entry:

- Supplier name
- Website or location
- Order date (in notes)

### Evaluating Suppliers

Use notes to track:

- Germination rates
- Seed quality
- Shipping experience
- Value for money

## Tips for Seed Inventory

1. **Update regularly** — Adjust quantities after each planting
2. **Check expiration** — Review monthly for expiring seeds
3. **Track suppliers** — Note which suppliers give best results
4. **Note germination** — Record actual germination rates
5. **Plan purchases** — Review inventory before ordering season
