# Quick Tour

This 5-minute tour introduces the main features of SowTempo. By the end, you'll understand how to navigate the app and use its core tools.

## The Main Interface

SowTempo has a simple layout with three main areas:

![Main Interface](/img/screenshots/interface-overview.png)

1. **Sidebar (left)** — Navigate between sections
2. **Content Area (center)** — Main workspace for the current section
3. **Context Panels (right)** — Details and inspectors (where applicable)

## Navigation Sections

Click any item in the sidebar to switch sections:

| Section           | Purpose                                  |
| ----------------- | ---------------------------------------- |
| Dashboard         | Overview, weather, quick stats           |
| Planting Schedule | Calendar view of all plantings           |
| Tasks             | To-do list auto-generated from plantings |
| Garden Layout     | Visual garden designer                   |
| Vegetables        | Plant database with 3,900+ varieties     |
| Garden Journal    | Notes and photos                         |
| Seasons           | Manage growing seasons                   |
| Seed Inventory    | Track seed stock                         |
| Analytics         | Charts and insights                      |
| Settings          | Preferences and configuration            |

Additional tracking features are grouped under **Tracking**:

- Harvests
- Soil Tests
- Pest Encounters
- Watering

## Dashboard Overview

The Dashboard is your garden's home base. It shows:

![Dashboard Tour](/img/screenshots/dashboard-tour.png)

- **Quick Stats** — Active plantings, GDD progress, tasks due, current temperature
- **Seasonal Context** — Current season, frost dates, timeline
- **Weather** — Current conditions and forecast
- **Today's Tasks** — What needs doing today
- **Upcoming Plantings** — What's coming up
- **Harvest Ready** — Plants approaching maturity

Click **Refresh** to update weather and recalculate recommendations.

## Creating Your First Planting

Let's add a planting to see how SowTempo works:

### Step 1: Open Planting Schedule

Click **Planting Schedule** in the sidebar.

### Step 2: Add a Planting

Click the **+ Add Planting** button.

### Step 3: Choose a Vegetable

1. Search for a vegetable (e.g., "tomato")
2. Select a variety (e.g., "Cherokee Purple")
3. Choose your planting method:
   - **Seed Indoors** — Start seeds inside, transplant later
   - **Direct Sow** — Plant seeds directly in the garden
   - **Transplant** — Plant purchased seedlings

![Add Planting](/img/screenshots/add-planting.png)

### Step 4: Review Dates

SowTempo automatically calculates recommended dates based on:

- Your frost dates
- The plant's frost tolerance
- Days to maturity

You can adjust any date by clicking on it.

### Step 5: Save

Click **Save** to add the planting to your schedule.

## The Calendar View

After adding plantings, the calendar shows colored blocks representing key dates:

![Calendar View](/img/screenshots/calendar-view.png)

- **Purple** — Seed start indoors
- **Green** — Transplant date
- **Orange** — Direct sow date
- **Red** — Harvest window

Click any block to view or edit that planting.

## Garden Layout Basics

The Garden Layout lets you design your physical garden:

### Drawing a Bed

1. Go to **Garden Layout**
2. Select the **Rectangle Tool** (press **B**)
3. Click and drag on the canvas to draw a bed
4. Release to finish

![Draw Bed](/img/screenshots/draw-bed.png)

### Other Drawing Tools

- **C** — Circle bed tool
- **P** — Polygon bed tool (click points, press Enter to finish)
- **V** — Select tool (move and resize beds)
- **A** — Plant placement tool

### Adding Plants to Beds

1. Select a bed
2. Choose the **Plant Placement** tool (**A**)
3. Click in the bed to add a placement
4. Select a vegetable from the picker
5. Adjust plant count and spacing pattern

![Plant Placement](/img/screenshots/plant-placement.png)

### Canvas Controls

- **Scroll** to zoom in/out
- **Drag** on empty space to pan
- **Cmd/Ctrl + Z** to undo
- **Cmd/Ctrl + Shift + Z** to redo

## Exploring the Plant Database

The Vegetables section contains nearly 4,000 varieties:

![Vegetables Database](/img/screenshots/vegetables-database.png)

### Browsing

- Click categories in the sidebar (Tomatoes, Beans, Greens, etc.)
- Switch between **Grid** and **List** view
- Click a variety for detailed information

### Searching

- Type in the search box to find specific plants
- Results show matches across all categories

### Filtering

Use filters to narrow results:

- **Frost Tolerance** — Hardy plants for early/late season
- **Days to Maturity** — Quick crops vs. long-season plants
- **Zone Compatibility** — Plants suited to your hardiness zone

## Recording a Harvest

Track your yields in the Harvests section:

1. Go to **Tracking > Harvests**
2. Click **Log Harvest**
3. Select the planting
4. Enter quantity and unit
5. Optionally rate quality and add notes
6. Save

![Log Harvest](/img/screenshots/log-harvest.png)

Over time, your harvest data feeds into Analytics to show productivity trends.

## Checking Tasks

SowTempo automatically generates tasks from your plantings:

![Tasks](/img/screenshots/tasks-view.png)

- **Today** — Tasks due today
- **Overdue** — Past due (marked in red)
- **Upcoming** — Future tasks
- **Weather** — Tasks with weather considerations

Click a task to see details or mark it complete.
